- 날짜 데이터는 `날짜형`으로 변경해야 여러가지 날짜 계산을 할 수 있다.
  
```python
df.info() # 잘 불러왔나 확인 했으면 정보 확인
```
### 문자형을 날짜형으로 변경
- `pd.to_datetime(컬럼, format = '날짜형식')`
 
|형식|설명|
|---|---|
|%Y|0을 채운 4자리 연도|
|%y|0을 채운 2자리 연도|
|%m|0을 채운 월|
|%d|0을 채운 일|
|%H|0을 채운 시간|
|%M|0을 채운 분|
|%S|0을 채운 초|

```python
df["Date1"] = pd.to_datetime(df["Date"], format = %Y-%m-%d)
```
